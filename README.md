# Sistem Manajemen Penjualan Toko

Starter project Kelompok 7 menggunakan Python, Tkinter, SQLite, dan konsep Object-Oriented Programming.

## Fitur awal

- CRUD produk
- CRUD pelanggan
- Transaksi penjualan
- Pemeriksaan dan pengurangan stok
- Pembayaran dan kembalian
- Poin pelanggan
- Riwayat dan total penjualan
- Enam class utama: Person, Kasir, Pelanggan, Produk, Penjualan, Toko

## Menjalankan aplikasi

```bash
python main.py
```

Jika perintah `python` tidak tersedia di Windows, gunakan:

```bash
py main.py
```

Database otomatis dibuat pada `data/toko.db`.
